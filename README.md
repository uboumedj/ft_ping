# 42 project: ft_ping

The goal of this project is to recode de `ping` command in C.

## How to use

* Compile the project with `make`
* Launch it with superuser rights (`sudo ./ft_ping [address/hostname]`)

## Options available

* `-c`: packet count
* `-i`: interval between packets
* `-t`: packet Time-to-live
* `-f`: flood mode
* `-h`: help (usage)
* `-v`: verbose mode
